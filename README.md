# Rural Society Simulator - 乡土社会多智能体基础框架
**Version: 0.0.0 | 基础社会结构搭建版**

## 项目目标
本项目旨在逐步搭建一个**由 AI 智能体组成的微型人工社会**，
让智能体在封闭社会结构中完成：
- 生命周期成长
- 社会关系互动
- 情感与偏执生成
- 自学习与行为演化

最终实现：**AI 智能体 + 情感偏执 + 自学习 + 社会演化** 的完整实验平台。

---

## 当前版本：0.0.0（基础搭建版）
本版本为**最底层基础框架**，核心目标：
- 搭建费孝通《乡土中国》风格的极简传统村落结构
- 实现村民基础属性：年龄、家庭、身份、成长过程
- 完成社会环境参数化、可配置、可稳定运行
- 不加入复杂 AI、不加入大模型、不加入强交互
- 把**社会结构、生命周期、环境参数、可视化**调稳

后续迭代路线：
- **0.1.0+**：搭建无交互基础智能体
- **1.0.0**：接入本地小模型
- **2.0.0**：接入带 Token 的完整模型

## 项目结构
```
rural-society-simulator/
├── config.yaml          # 社会与人格参数配置
├── main.py              # 程序入口
├── src/
│   ├── agent.py         # 村民基础智能体（生命周期+人格）
│   ├── model.py         # 村落社会模拟器
│   ├── visualize.py     # 结果可视化
│   └── utils/
│       └── yaml_loader.py  # 配置加载
└── README.md
```

## 核心功能（v0.0.0）
- 封闭乡土社会结构
- 村民年龄增长与生命周期
- 基础五维人格参数

- 模拟过程数据记录
